# 🥘 BestRecipes — сайт с любимыми рецептами 

BestRecipes — это удобный веб-сервис для хранения, просмотра редактирования ваших кулинарных шедевров. Делитесь любимыми блюдами, открывайте новые рецепты и управляйте своей личной кулинарной коллекцией!

## 🚀 Возможности 
### 🔍 Просмотр рецептов 
Исследуйте коллекцию рецептов от других пользователей — от быстрых закусок до сложных десертов.

### ➕ Добавление собственного рецепта 
Создайте карточку рецепта с описанием, ингредиентами и инструкциями.

### ✏️ Редактирование рецептов
Обновите рецепт, если нашли более вкусный вариант или добавили секретный ингредиент!

### 🗑️ Удаление рецептов
Больше не готовите это блюдо? Просто удалите его из своей коллекции.

### 🖼️ Интерфейс
Простой, понятный и адаптивный интерфейс: всё, что нужно — под рукой.
